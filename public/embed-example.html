Exemplo de como embedar o sistema em outros websites

Como usar o iframe

Você pode embedar o sistema de consulta de hóspedes em qualquer website usando um iframe. Aqui estão alguns exemplos:

<!-- Versão completa --> <iframe src="YOUR_APP_URL" width="100%" height="600"></iframe> <!-- Versão compacta para iframe --> <iframe src="YOUR_APP_URL/iframe" width="400" height="300"></iframe> <!-- Com parâmetro de idioma --> <iframe src="YOUR_APP_URL?lang=es" width="100%" height="600"></iframe> <iframe src="YOUR_APP_URL/iframe?lang=pt" width="400" height="300"></iframe>

Exemplos em funcionamento:

Versão Completa

Versão Compacta (Iframe)

Configurações recomendadas:

  • Largura: Mínimo 300px para a versão compacta
  • Altura: 300px para a versão compacta, 600px para a versão completa
  • Responsivo: Use width="100%" para layouts responsivos

URLs disponíveis:

  • YOUR_APP_URL - Versão completa (Português)
  • YOUR_APP_URL?lang=es - Versão completa (Espanhol)
  • YOUR_APP_URL/iframe - Versão compacta (Português)
  • YOUR_APP_URL/iframe?lang=es - Versão compacta (Espanhol)
